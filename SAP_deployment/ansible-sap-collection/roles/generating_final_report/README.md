# genarting_final_report
This role is for generating the final report.

## Overview
The role involves a number of steps:

* create folder to store outputs
* merge the files and provide delimiter
* tidy up md-collection folder

## Role variables
The variable to be used within this role are all defined at playbook level.

## Example playbook
```yaml
---
- hosts: localhost
  roles:
    - genarting_final_report
```
