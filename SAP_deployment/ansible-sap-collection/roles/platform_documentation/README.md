# platform_documentation
This role is used to generate the infra structure.

## Overview
The platform documentation involves a number of steps:

* tasks for azure
* tasks for aws
* tasks for gcp

## Role variables
The variables to be used within this role are all defined at playbook level.

## Example playbook
```yaml
---
- hosts: localhost
  roles:
    - platform_documentation
```
